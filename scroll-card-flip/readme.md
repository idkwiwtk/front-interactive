# 스크롤 카드 플립 애니메이션

참고 사이트: <https://rara.ritsumei.ac.jp/>

스크롤 시, 카드 섹션에서 sticky 박스로 전환되며, 스크롤 시, sticky 영역 내에서, 카드가 오른쪽에서 나타나며 뒤집기 애니메이션이 적용된다.

- 스크롤 애니메이션이 시작되는 구간을 먼저 정의한다.
- 각 카드 별로 애니메이션이 시작되고 종료되는 구간을 계산한다.
- 카드 갯수 별로 계산할 수 있도록 step 을 조정한다.
- 각 카드의 애니메이션 시작 전, 진행 중, 카드 뒤집기 구간, 모든 애니메이션이 종료되는 구간 별, css transform 속성을 조정한다.
- 카드의 x 축 이동은 translateX, 카드의 flip 진행은 rotateY를 사용하고, 스크롤 되는 구간의 퍼센트를 계산하도록 한다.
